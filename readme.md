# OpenSV 插件包
#### 本仓库为OpenSV的插件包。
##### 目前插件支持
###### 一、Opencv
1.图像读取

2.图像灰度

3.图像二值化

4.图像显示

5.图像保存

6.图像BGR通道分离

7.图像HSV通道分离

8.图像高斯模糊

9.图像翻转

9.图像腐蚀

9.图像膨胀


二、Mediapipe

1.姿态检测

2.手势检测
